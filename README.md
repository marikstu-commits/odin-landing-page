# 🌿 Serenity Massage Parlor

An elegant and responsive landing page developed for **Serenity Massage Parlor**, focused on wellness, holistic massage techniques, and relaxation. This project was built from scratch using HTML5 semantic best practices and modern CSS3 (Flexbox).

---

## 🎨 Visual Identity & Color Palette

The website design is fully inspired by the logo's identity, utilizing earthy and natural tones to convey calmness and sophistication:

| Element | Matching Color | Hexadecimal |
| :--- | :--- | :--- |
| **Main Background** | Off-White / Soft Cream | `#F6F4EB` |
| **Borders & Details** | Sage / Olive Green | `#8BA280` |
| **Contours & Leaves** | Dark Olive Green | `#3A5333` |
| **Text & Headings** | Dark Anthracite Gray | `#1F2937` |

---

## 🚀 Features

* **Semantic Structure:** Proper use of tags such as `<header>`, `<nav>`, `<section>`, and `<footer>` for improved accessibility and SEO.
* **Responsive Layout:** Flexbox-based sections that adapt elegantly to different screen sizes.
* **Optimized Images:** Implementation of `object-fit` techniques to prevent distortion on logos and project images.
* **Clean Navigation:** Structured links ready for future Services, About, and Contact Us pages.
